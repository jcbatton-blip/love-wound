# MEMBER250 Stripe Promo Code Setup

## Quick Setup (2 minutes)

1. Go to https://dashboard.stripe.com/coupons
2. Click **"Create coupon"**
3. Configure:
   - **Name:** Inner Circle Member Discount
   - **ID:** MEMBER250
   - **Type:** Fixed amount
   - **Amount off:** $100.00 USD
   - **Duration:** Forever
   - **Applies to:** Specific products → Select "Mirror Session"
4. Click **"Create coupon"**
5. Click **"Create promotion code"** → Use code: MEMBER250
6. Done!

## How It Works

- Regular Mirror Session price: $350
- With MEMBER250 code: $250
- Members enter code at checkout
- Discount applies automatically

## Testing

After creating the code:
1. Go to your Services page
2. Click "Book A Session" for Mirror Session
3. At checkout, enter promo code: MEMBER250
4. Price should drop from $350 to $250
5. Complete test purchase or cancel

## Member Communication

Include in welcome email:
"Members get Mirror Sessions at $250 (vs $350 regular price). Use code: MEMBER250 at checkout."

# Inner Circle Calendar Setup Guide

The Google Calendar API is having timezone issues, so here's the manual setup (takes 3 minutes):

## Step 1: Create Recurring Event for 1st Tuesday

1. Go to https://calendar.google.com
2. Click **"Create"** button (top left)
3. Fill in details:
   - **Title:** `Inner Circle Group Session (1st Tuesday)`
   - **Date:** February 4, 2026
   - **Time:** 5:00 PM - 6:30 PM
   - **Timezone:** Eastern Time
   - Click **"Does not repeat"** dropdown → Select **"Custom..."**
   - Set to: **"Monthly on the first Tuesday"**
   - **Location:** [Paste your Zoom link here]
   - **Description:**
     ```
     Inner Circle monthly group coaching session - Group A
     
     Pattern hunting in real time. Max 12 members.
     
     Members will be added as guests to this recurring event.
     ```
   - **Guests can:**
     - ☐ Modify event (unchecked)
     - ☐ Invite others (unchecked)
     - ☑ See guest list (checked)
4. Click **"Save"**

## Step 2: Create Recurring Event for 3rd Tuesday

1. Click **"Create"** again
2. Fill in details:
   - **Title:** `Inner Circle Group Session (3rd Tuesday)`
   - **Date:** February 18, 2026
   - **Time:** 5:00 PM - 6:30 PM
   - **Timezone:** Eastern Time
   - Click **"Does not repeat"** dropdown → Select **"Custom..."**
   - Set to: **"Monthly on the third Tuesday"**
   - **Location:** [Same Zoom link]
   - **Description:** [Same as above]
   - **Guests can:** [Same settings]
3. Click **"Save"**

## Step 3: When Someone Joins Inner Circle

1. Open one of the recurring events in Google Calendar
2. Click **"Edit event"** → **"Edit all events"** (to add them to all future sessions)
3. Click **"Add guests"**
4. Enter their email address
5. Click **"Save"** → **"Send"** (they'll get invite + all future reminders)

## Notes

- When you hit 12 members in Group A, create Group B with different days (2nd & 4th Tuesday)
- Same Zoom link can be used for both recurring events
- Members get automatic reminders 1 day before and 1 hour before each session
- You can see who's invited by opening any event and checking the guest list
